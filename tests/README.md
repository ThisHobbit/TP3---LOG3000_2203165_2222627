Ce module contient une suite de tests unitaires pour vérifier le bon fonctionnement des fonctions arithmétiques de base définies dans utils/operators.py.

Les tests utilisent le module standard unittest de Python.

Chaque fonction (add, subtract, multiply, divide) est testée sur :
    - des cas normaux (valeurs positives et négatives) ;

    - des cas limites comme :

        - opérations avec zéro ;

        - gestion des erreurs (division par zéro) ;
